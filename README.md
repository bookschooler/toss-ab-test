# 토스증권 해외주식 소수점 구매 A/B 테스트

> **Toss Securities** · 입력창 문구 변경이 구매 전환율과 의사결정 속도에 미치는 영향 검증

---

## 프로젝트 개요

토스증권 해외주식 소수점 구매 페이지에서 입력창 문구 A/B 변경 실험을 설계·분석한 포트폴리오 프로젝트입니다.  
"얼마나 구매할까요?"(A) vs "얼마치 구매할까요?" + 예상 주수 실시간 표시(B)의 효과를 통계적으로 검증했습니다.

| 항목 | 내용 |
|---|---|
| 실험 기간 | 4일 (최소 검출 가능 효과 기준 산정) |
| 실험 대상 | 해외주식 소수점 구매 진입 유저 35,000명 |
| 유의수준 α | 5% / 검정력 80% |
| 최소 샘플 | 그룹당 7,850명 |

---

## 핵심 결과

| 지표 | A군 (대조) | B군 (실험) | 변화 | p-value |
|---|---|---|---|---|
| 구매 전환율 (CVR) | 9.41% | 11.93% | **+2.52%p** ✅ | 0.0013 |
| 의사결정 속도 (첫 경험자) | 27.5초 | 13.8초 | **-50%** ✅ | < 0.0001 |
| 퍼널 입력창 잔존율 | 27.3% | 46.2% | **+18.9%p** ✅ | — |

→ **B안 전면 적용 권고. 예상 추가 전환 882명/4일 (연간 ~80,000명 추산)**

---

## 분석 방법론

```
데이터 생성 → SRM 검정 → EDA → 통계 검정 → 퍼널 분석 → 결과 보고
```

| 단계 | 방법 | 이유 |
|---|---|---|
| 데이터 검증 | SRM (Sample Ratio Mismatch) 검정 | 실험 무결성 확인 (p=0.73, 통과) |
| 이상치 처리 | Winsorizing (상위 5%) | 의사결정 시간 right-skewed 분포 보정 |
| 전환율 비교 | Chi-square test | 이진 범주형 비교 |
| 속도 비교 | Mann-Whitney U test | 정규성 불충족 → 비모수 검정 |
| 추가 검증 | 퍼널 분석 (4단계) | 이탈 구간 식별 |

## 주요 인사이트

**희석 효과(Dilution Effect):** 전체 유저 기준 DV 차이가 미미한 이유 — 재방문 유저가 문구와 무관하게 빠르게 행동해 효과를 희석. 첫 경험자 분리 분석으로 실제 효과 포착.

**성공 조건 이중 충족:** DV 단축만으로는 "빠르게 포기"한 케이스도 포함됨 → CVR 동시 개선 확인 후 진정한 UX 개선으로 판정.
